Add a new AI provider

1. Add entries to config/models.yaml with provider: openai (or any name).
2. Create model_router/providers/openai.py, subclass BaseProvider, implement execute().
3. Register in router.py: self._providers["openai"] = OpenAIProvider().
4. Click Reload above.
